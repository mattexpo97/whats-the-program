# What's the program?

## Live demo

You can see a live demo [here](https://mattexpo97.github.io/whats-the-program/)


## Packages used

Nuxt 3
VueUse
Naive UI
UnoCSS
Pinia
Pinia Persisted Storage
Unplugin icons


## Setup

Make sure to install the dependencies:

```bash
# npm
npm install

# pnpm
pnpm install

# yarn
yarn install
```

## Development Server

Start the development server on `http://localhost:3000`:

```bash
# npm
npm run dev

# pnpm
pnpm run dev

# yarn
yarn dev
```

## Production

Build the application for production:

```bash
# npm
npm run build

# pnpm
pnpm run build

# yarn
yarn build
```

Locally preview production build:

```bash
# npm
npm run preview

# pnpm
pnpm run preview

# yarn
yarn preview
```
